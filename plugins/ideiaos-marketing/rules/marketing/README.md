# Marketing Best Practices

Este diretorio contem 22 best-practices absorvidas do OpenSquad (MIT) via quarentena ECC.
Cada arquivo define as regras de operacao de um agente de marketing para um formato ou disciplina especifica.
O Architect carrega apenas os arquivos relevantes para o squad sendo construido.

| ID | Nome | Arquivo | Quando Usar |
|----|------|---------|-------------|
