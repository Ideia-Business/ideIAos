```yaml
---
name: evolve
```

Linha `# SOURCE: IdeiaOS v2`. Corpo PT-BR (~90 linhas):

- **Quando:** `/instinct-status` mostra instincts ≥0.7; consolidação periódica.
- **Pipeline de promoção:**
  1. Varrer `~/.ideiaos/instincts/{project,global}/*.md`; selecionar os com `confidence >= 0.7`.
  2. Para cada maduro, decidir destino:
     - **Regra de comportamento** (stack-agnóstica, prescritiva — "sempre X antes de Y") → `source/rules/common/` ou `source/rules/<stack>/` (propaga às IDEs via build-adapters). Header de rule do projeto (`<!--SOURCE: IdeiaOS v2 | kind: rule | targets: ...-->` é o padrão das rules existentes — confirmar olhando uma rule atual em `source/rules/` e seguir o MESMO formato dela; rules NÃO usam o header markdown de skill).
     - **Learning de projeto** (síntese curada cross-projeto) → vault Obsidian: usar `$VAULT/_Templates/Learning.md` se existir; criar `$VAULT/Learnings/<Título humano>.md` (nome legível, não slug — graph view), frontmatter `applies_to`, tags `#stack/...` `#category/...`, cross-refs como wiki-links `[[...]]` incluindo `[[00 Index]]`. EXATAMENTE como `extract-learnings` Passo 4b descreve.
       ```bash
       VAULT="$HOME/Library/Mobile Documents/iCloud~md~obsidian/Documents/Ideia Business - Second Brain"
       ```
       Se o vault não existir no path (máquina sem iCloud) → pular promoção ao vault, deixar o instinct marcado `promoted: pending` e avisar.
  3. Após promover, marcar o instinct de origem: `promoted: true` + `promoted_to: <path>` + `promoted_at: <data>` no frontmatter (não deletar — rastreabilidade; ou mover para `~/.ideiaos/instincts/_promoted/` se preferir manter o ativo enxuto — documentar a escolha).
- **Curadoria (decay + dedup):**
  - **Dedup:** instincts com `slug(trigger)` equivalente ou ações redundantes → fundir (somar `evidence_count`, manter maior confidence, registrar `updated`).
  - **Decay:** instinct não reforçado (sem `updated` recente — heurística: muitas sessões/observações novas sem que ele tenha sido reativado) → reduzir confidence ~0.1. Se cair abaixo de ~0.2, arquivar (`~/.ideiaos/instincts/_archive/`). NUNCA deletar silenciosamente sem registrar.
- **Privacidade:** ao promover ao vault, reaplicar memory-hygiene (Regra 1) — nenhum secret/credencial/conteúdo literal na nota. Abstrair.
- **Saída:** `🧬 Evolve: N instincts promovidos (X→vault, Y→rules), M deduplicados, K decay/arquivados.`
- **Anti-padrões:** não promover instinct <0.7; não despejar no vault (curadoria, não dump — mesma disciplina do extract-learnings); não duplicar Learning já existente no vault (buscar antes); não deletar instinct sem rastro.

```bash
if [ -f "$HOOK_FILE" ]; then

    ok "Hook observe-tool-use já está na versão mais recente"

else

SNIPPET
```
