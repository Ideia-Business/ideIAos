  PATTERNS = re.compile(r'curl|wget|nc |scp |ssh |enableAllProjectMcpServers|ANTHROPIC_BASE_URL')

Substituir `nc ` por `\bnc\b`:

  PATTERNS = re.compile(r'curl|wget|\bnc\b|scp |ssh |enableAllProjectMcpServers|ANTHROPIC_BASE_URL')

Justificativa: `\b` é word boundary em Python re — rejeita `function`, `sync`, `async`,
`truncate`, `synchronize` etc. (todos contêm "nc" como substring, não como token isolado).
Netcat real (`nc localhost 4444`) ainda é detectado pois `nc` é palavra isolada.

NÃO alterar nenhum outro check. NÃO alterar lógica de PASS/WARN/FAIL. NÃO usar `<!-- -->`.
  </action>

    <automated>python3 -c "
import re
P = re.compile(r'curl|wget|\bnc\b|scp |ssh |enableAllProjectMcpServers|ANTHROPIC_BASE_URL')
# deve ser falso (sem match)
false_pos = ['function connect()', 'async function', 'truncate(buffer)', 'synchronize()', 'bounce()']
# deve ser verdadeiro (match real)
true_pos  = ['nc 192.168.1.1 4444', 'nc -lvp 9001', 'bash -c nc target']
ok = all(not P.search(s) for s in false_pos) and all(P.search(s) for s in true_pos)
print('PASS' if ok else 'FAIL')
for s in false_pos:
    m = P.search(s)
    if m: print(f'FALSE POSITIVE: {s!r} matched {m.group()!r}')
for s in true_pos:
    if not P.search(s): print(f'FALSE NEGATIVE: {s!r}')
"
</automated>
