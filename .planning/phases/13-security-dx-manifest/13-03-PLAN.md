Criar scripts/apply-to-all-projects.sh com a seguinte especificação:

**Header:**
```
#!/usr/bin/env bash
# =============================================================================
# apply-to-all-projects.sh — propaga setup.sh --project-only a todos os repos ~/dev/
#
# Detecta repositórios git em ~/dev/* (1 nível de profundidade — não recursivo).
# Exclui o próprio IdeiaOS e diretórios sem .git.
# Dry-run por padrão: lista sem executar. Sem a flag: executa em cada repo.
#
# Uso:
#   bash scripts/apply-to-all-projects.sh                 # dry-run (seguro)
#   bash scripts/apply-to-all-projects.sh --apply         # executa em todos
#   bash scripts/apply-to-all-projects.sh --apply --only nfideia,lapidai  # filtro
#
# Exit: 0 se tudo OK; 1 se qualquer repo falhou (modo --apply).
# SOURCE: IdeiaOS v2
# =============================================================================
set -uo pipefail
```

**Variáveis e parsing:**
```bash
SETUP_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")/.." && pwd)"
DEV_DIR="${DEV_DIR:-$HOME/dev}"
DRY_RUN=1; ONLY_FILTER=""
for arg in "$@"; do
  case "$arg" in
    --apply) DRY_RUN=0 ;;
    --only) shift; ONLY_FILTER="${1:-}" ;;
    --dry-run) DRY_RUN=1 ;;
  esac
done
```

**Cores** (mesmas do idea-doctor.sh):
```bash
GREEN='\033[0;32m'; YELLOW='\033[1;33m'; CYAN='\033[0;36m'; BOLD='\033[1m'; NC='\033[0m'
ok()   { echo -e "${GREEN}  ✓${NC} $*"; }
skip() { echo -e "${CYAN}  ⊙${NC} $*"; }
warn() { echo -e "${YELLOW}  ⚠${NC}  $*"; }
```

**Detecção de repos:**
Iterar sobre `"$DEV_DIR"/*/`: incluir apenas diretórios que contenham `.git/` E que não sejam o
próprio IdeiaOS (`[ "$(realpath "$d")" = "$SETUP_DIR" ]`).
Se `--only` foi fornecido, filtrar por nome: `[[ ",$ONLY_FILTER," == *",$(basename "$d"),"* ]]`.

**Modo dry-run** (DRY_RUN=1):
Imprimir cabeçalho "Projetos detectados em $DEV_DIR (dry-run — use --apply para executar):"
Para cada repo: `skip "$(basename "$d")  →  $d"`
Imprimir rodapé com contagem: "N projeto(s) detectado(s). Rode com --apply para executar."
Exit 0.

**Modo --apply** (DRY_RUN=0):
Para cada repo:
  `ok "Aplicando setup.sh --project-only em $(basename "$d")..."`
  `bash "$SETUP_DIR/setup.sh" --project-only "$d" && ok "$(basename "$d") OK" || warn "$(basename "$d") terminou com erro (ver acima)"; ERRORS=$((ERRORS+1))`
Rodapé: "N projeto(s) processado(s). Erros: $ERRORS."
Se ERRORS > 0: exit 1, senão exit 0.

**Convenções:**
- Fail-silent em subshells (`2>/dev/null` em `realpath` para compatibilidade).
- NÃO usar jq. NÃO usar `<!-- -->`.
- `chmod +x` não é necessário — o pre-commit hook check-readme-sync audita `scripts/*.sh` por nome.
