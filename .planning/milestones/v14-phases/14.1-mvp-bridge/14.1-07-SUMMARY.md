# Phase 14.1 Plan 14.1-07: TtT Bridge Harness Summary

`ttt-baseline.sh --mode=bridge` agora grava medicao REAL (modo literal `bridge`, segundos reais via `node:sqlite` sobre o read-model) para as 3 jornadas, e `ttt-median.sh --mode=bridge` seleciona essas linhas e crava o gate A2 (<10s) por exit-code — medianas Bridge J1=0.026s / J4=0.025s / J2=0.025s, baseline terminal 14.0-03 preservado.

## What Was Built

### Task 1 — `ttt-baseline.sh --mode=bridge` mede REAL (substitui placeholder)
O bloco `--mode=bridge` que antes gravava `bridge-placeholder\t0.000` agora:
- Cronometra (`date +%s%N` antes/depois) a jornada **respondida via a Bridge** — `node:sqlite` (`DatabaseSync`, readOnly) abre `~/.ideiaos/console/read-model.db` (a MESMA fonte que os endpoints `/fleet` e `/vault` de 14.1-01 servem) e roda a SELECT da jornada.
- Grava `printf '%s\t%s\t%s\t%s\n' "$JORNADA" "bridge" "$SECONDS" "$NOW_EPOCH"` — modo **literal `bridge`**, formato TSV 4-col intacto.
- **3 jornadas** (queries determinísticas, read-only):
  - **J1** frota-saudavel: `COUNT(machine)` + `COUNT(daemon_status)` (fonte de `/fleet`).
  - **J4** chave-existe+idade: `COUNT(*)`, `SUM(present)`, `MAX(file_mtime_epoch)` em `api_key` — **METADATA-ONLY**; o schema nao tem coluna `value` e a query nunca a introduz (credential-isolation).
  - **J2** pronto-pra-tag: `soak_satisfied = COUNT(DISTINCT host)>=2 AND (MAX(epoch)-MIN(epoch))>=86400` em `soak_heartbeat` com `idea_doctor`+`regression` PASS — span sobre **EPOCHS GRAVADOS** (delta), nao wall-clock.
- `--dry-run` intacto: valida estrutura, exit 0, **nao grava** linha bridge. Modo terminal intacto.

### Task 2 — `ttt-median.sh --mode=bridge` SELECIONA bridge + gate A2
- Parse de `--mode=bridge|terminal` (default terminal), preservando o arg posicional de TSV custom e o gate de TSV ausente/vazio (exit 1).
- `median_for_journey` mode-aware:
  - `bridge` → SELECIONA somente `$2=="bridge"`.
  - `terminal` → EXCLUI `bridge-placeholder`, `bridge-dry-run` **e `bridge`** (mantem o baseline terminal puro).
- Saida 2-col `<jornada>\t<mediana>` por J1/J4/J2 — o gate A2 `awk '$2>=10{f=1} END{exit f}'` sai 0.
