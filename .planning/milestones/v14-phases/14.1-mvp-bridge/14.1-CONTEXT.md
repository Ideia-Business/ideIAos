<domain>
## Phase Boundary

**v14.1 — MVP Bridge** is the first phase that delivers *user-visible value* on top of the v14.0 substrate (which already shipped: `idea-doctor --json`, ref `cockpit`, `ideiaos-agentd` collector, `console-ingest`→SQLite read-model, SPA scaffold at `apps/cockpit/`, idea-doctor §15). It is a **vertical slice, read-only w.r.t. production, no login, served on loopback `http://127.0.0.1`**.

**Requirement IDs this phase MUST satisfy:** **R14-05** (MVP Bridge read-only UI + ⌘K + Flight Recorder v0) and **R14-06** (Zero-Leak release gate + Time-to-Truth measurement harness). Every PLAN.md must carry one of these IDs in its `requirements` field.

**IN scope:**
- Read metadata from the `cockpit` ref / read-model DB / disk and render it.
- Execute **local reversible** verbs only (pause/resume autosync, run idea-doctor, re-seal security freshness, kickstart a local daemon).
- The Zero-Leak invariant test (release gate) and the Time-to-Truth (TtT) measurement harness.

**OUT of scope (pushed to v14.2+/v14.4):** Constelação, Sinapse, the full Pulso, the Atalaia *screen* (only ⌘K verbs land here), Time-Travel (full), Copiloto; **any** production-mutating verb (`rotate`/`revoke`/`deploy`); **any** cross-machine command; **any** login/RBAC. Those are v14.4, behind the security `/spec` + threat-model gate.

## Deliverables (doc 02 §3, amended by doc 71 §5)

1. **Overview screen** (bento grid): System Pulse (local-live hero), cards Frota / Segurança / Releases-SOAK (lean: "PRONTO PARA TAG" + `v## n/2 ✓ span x/1d`, **no big countdown** — it cedes the slot) / Atenção-Agora, **+ Flight Recorder v0 band (6×1, new)**.
2. **Frota screen**: per-machine card + dense heartbeat table + version drift (string-equality, never semver).
3. **Cofre-Espelho screen**: var × project matrix, **metadata-only** (no UI control reads/copies/writes/rotates a value), doctrine banner, celebrated empty state.
4. **Command Palette ⌘K**: fixed allowlist of local reversible verbs, inline result.
5. **Flight Recorder v0**: deterministic replay of the real `gsd`-pin flip-flop in `versions.lock` via `git show <sha>:versions.lock` — a single-series time-tape (the structural "wow"). Reads the **local IdeiaOS git only** (does not even need the `cockpit` ref).
6. **Zero-Leak invariant test** (`npm run test:zeroleak`, exit-code release gate).
7. **TtT measurement harness** (terminal baseline N≥5 + Bridge median <10s).
8. **idea-doctor §15** consumes the real console state (already exists from v14.0 — verify/extend, do not regress its ANSI output).

### Stack (zero new choices — doc 02 §2)

Single file `~/.ideiaos/console/read-model.db` (disposable; `rm && console-ingest` rebuilds from the `cockpit` ref). The SPA reads it over a **loopback HTTP server already shipped in v14.0** at `apps/cockpit/server/read.js` (bound `127.0.0.1:3073`). v14.1 EXTENDS this server (e.g. the ⌘K command channel), not a new one. _[Corrected from doc 02's stale `better-sqlite3` per RESEARCH.md RQ1 + PATTERNS — verified in the shipped code.]_
- **No new dependency for data access** (uses the zero-dep `node:sqlite` built-in). The ONE sanctioned new UI dep is **`cmdk`** (the shadcn command-palette primitive, already used in nfideia) for the ⌘K palette. Flight Recorder v0 is **hand-rolled SVG** (no chart dep; mirror `HealthScore.jsx`). No auth lib. Prefer native (Enforce Simplicity + `credential-isolation`).
- Reuse components: `KPICard`, `AppLayout`, `AppSidebar`, `NotificationBell` (nfideia); `HealthScore`, `TrendChart` (`.aiox-core/.../health-dashboard`). The pattern-mapper maps exact analogs.

### Zero-Leak (R14-06; spec req "Isolamento absoluto"; doc 78 §1) — the P0 release gate
- A secret **value** in **any** surface is a P0 that **blocks merge** (gate, exit-code — never advisory, never the Read tool).
- **7 surfaces** scanned, each materialized to a file first then scanned by exit-code: S1 ref snapshot (`git show cockpit:snapshots/<MID>.json`), S2 `read-model.db .dump`, S3 schema (`PRAGMA table_info(api_key)` — assert NO `value` column), S4 serialized React state, S5 rendered DOM, S6 loopback network bodies, S7 logs.
- Detector = (a) literal known-key regex (incl. `eyJ…\.eyJ` JWT for `service_role`/anon, `sk-`, `sk-ant-`, `gh[pousr]_`, Slack, long base64) + (b) Shannon entropy ≥4.0 with a **name/shape-based** allowlist (`machine_id`=sha256, commit SHAs, public `supabase_project_id`, `input_hash`, audit-log hashes) — never value-based.
- **Positive structural proof** beats negative scan: `api_key` table has NO `value` column AND no credential object carries a `value` key.
- **Dogfood-veneno (mandatory):** a planted `sk-ant-FAKEKEY…` in a `/tmp` snapshot MUST make `test:zeroleak` exit ≠0 — otherwise the gate is theater.

### Time-to-Truth (R14-06; spec req "Verdade verificável"; doc 78 §2)
- 3 journeys: **J1** "is the fleet healthy?", **J4** "does key X exist and how old?" (never the value), **J2** "ready to tag?" (`soak_satisfied` = ≥2 machines + span≥1d over recorded epochs, NOT wall-clock).
- Baseline measured via terminal (greps/awk/launchctl, `date +%s.%N`) N≥5 per journey → `ttt-baseline.tsv`; median via `ttt-median.sh` (pure bash). **Note: the v14.0 plan `14.0-03` already established the terminal baseline (N≥5/journey)** — v14.1 measures the **Bridge** side and proves median <10s vs baseline (gate: `ttt-median.sh --mode=bridge | awk '$2>=10{f=1} END{exit f}'`).
- **Trust-Rate = verify-against-disk-now** (not cache): `--verify` recomputes from `git show cockpit:...` at question time; UI shows "verificado há Xs"; divergence flagged stale. Target 100%.

### Command Palette ⌘K allowlist (R14-05; spec req "Comando local reversível"; doc 77 §B)
- **Default-deny, explicit allow.** Every verb must pass 4 binary criteria: LOCAL (process IPC to the **own machine's** agentd, never git/cross-machine), REVERSIBLE (a sanctioned inverse verb exists locally), NON-PRODUCTION-MUTATING, NON-@devops-authority.
- **In allowlist (v14.1):** B1 Pausar autosync (`autosync-pause.sh on`), B2 Retomar autosync (`autosync-pause.sh off`), B3 Re-selar segurança (`check-security-freshness.sh --record PASS @security-reviewer`), B4 Forçar sync (`launchctl kickstart -k gui/$UID/com.ideiaos.gitautosync`), B5 Kickstart daemon X (`launchctl kickstart` of `envsync`/`refresh-ai-security`/`cockpit`), B6 Rodar idea-doctor (`idea-doctor`, read-only).
- **Armar-antes-de-disparar** (explicit confirmation) for destructive-but-reversible verbs (B1, B3).
- **Forever OUT** (doc 77 §B.2): `git push`/`gh pr` (@devops-exclusive — at most *generate* the command), MCP add/remove (@devops), `rotate`/`revoke`/`deploy` (production mutation — v14.4 gated), any cross-machine command (v14.4), `reveal`/copy a secret value (NEVER — `credential-isolation`), arbitrary `exec`/shell in agentd (NEVER — Excessive Agency), mass `revoke` (NEVER — structural DoS). The ⌘K never invents a verb — it only exposes existing sanctioned CLI (Constitution: CLI First).

### Flight Recorder v0 (R14-05; doc 71 — the structural "wow")
- ONE reconstruction, read-only: the `gsd`-pin flip-flop (`1.36.0 → 1.1.0 → 1.36.0 → 1.1.0`, reverted 3× over 18 days). Source = **local IdeiaOS git** (`git log --format='%H|%cI|%s' -- versions.lock`; `git show <H>:versions.lock | grep '^gsd='`). 13 commits, 1 key, 1 dimension.
- Actor classification is **deterministic string-match**: `^wip: autosync` → daemon, else human (blueprint §9). Host from the `(MacBook-Air-2)`/`(Mac-mini-de-Gustavo)` suffix.
- UI = a horizontal **two-level step-line** SVG tape (`1.36.0` top, `1.1.0` bottom), color-by-state-never-decoration (amber `--status-warning` on a daemon-reversal node; discreet green `--status-success` `✓` on the stable end; gold reserved for frame/selection only). Hover → mono tooltip `06-08 · gsd=1.36.0 · daemon (Mac-mini) · "wip: autosync"`. Click → side panel `git show --stat`. Draw-left-to-right entrance ~600ms behind `prefers-reduced-motion`.
- **LAW (exit-code) vs INTERPRETED:** the pin value/order/daemon-reversal are LAW (`git show` + string-match); the *narrative* ("daemon reverted human", "redux inverted semver") is a **labeled annotation**, never asserted as fact. `<absent>` where git has no `gsd=` — never invented.
- **Invariant test (gate):** `npm run test:recorder` re-derives the tape from git and compares SET-to-SET with the render → exit 0 if identical, exit 1 if the UI diverged. Plus ≥1 amber reversal node must appear (the real flip-flop).

### Honest freshness (spec req "Frescor honesto"; doc 02 §4)
- System Pulse animates ONLY on the **local heartbeat** (file-watch of own machine's snapshot, ~1–5s); remote nodes show honest age ("último sinal há Xh"), never simulated continuous flow. Critical signal → red/arrhythmic state, and **color is never the only indicator** (text label — WCAG).

### Per-product health honest sub-signal (spec req "Saúde por produto"; doc 02 §4)
- idea-doctor doesn't run on Lovable products → that sub-signal renders `n/a`, never a fabricated score; aggregate counts it as neither pass nor fail. Single-operator reality (`gustavo@` only) → P1/P2 personas labeled "aguardando segundo ator", not filled with fake data.

### @devops authority respected (spec req "Respeito à autoridade")
- The Cockpit NEVER executes `git push`, `gh pr`, or MCP add/remove/configure. When relevant, it at most **generates the command** for @devops.

### Verification discipline (`antifragile-gates`)
- Two regimes: artifact-of-file (`test -s`/exit-code is LAW, never the Read tool) and runtime/UI (render + screenshot + a11y-tree with explicit criteria, `frontend-visual-loop`). Where exit-code exists, it is law. Git tests run in `/tmp` sandbox with autosync paused (`verify-guards-in-sandbox-not-live-repo`).

### Claude's Discretion (planner decides)
- Wave/plan decomposition (likely vertical slices per pillar + the two R14-06 gates), file/component names within `apps/cockpit/src/`, exact React state shape, how the SPA reads SQLite (build-time fixture vs a tiny local read endpoint on loopback — must stay loopback-only and Zero-Leak-clean), the IPC mechanism between the SPA and the local agentd for ⌘K verbs (must be local process IPC, not git, not a public port).
