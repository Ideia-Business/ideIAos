**Source:** `schema.sql:58-67` (`api_key` NO `value` column) + `collect.js readEnvKeys:160-186` (RHS of `=`
discarded). **Apply to:** Cofre-Espelho UI (renders metadata only), Zero-Leak S3 (assert no `value` col),

`sqlite3 db 'PRAGMA table_info(api_key)' | cut -d'|' -f2 | grep -qiw value && exit 1`.

### Antifragile gate exit-code (every `.sh` gate/harness)
