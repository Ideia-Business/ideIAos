# Phase 14.1 Plan 14.1-03: Frota + Cofre-Espelho Summary

As duas telas de pilar restantes da Wave 3 do Cockpit: **Frota** (card por máquina via reuso de `MachineCard` + tabela densa de heartbeat + version-drift por igualdade de string) e **Cofre-Espelho** (matriz var × project metadata-only com banner de doutrina e empty-state celebrado). Ambas consomem os endpoints `/fleet` e `/vault` de 14.1-01 e preenchem os stubs que 14.1-02 deixou — **sem editar App.tsx** (seam de file-ownership da Wave 3 intacto).

## Tasks

### Task 1 — Frota (`apps/cockpit/src/pages/Frota.tsx`)
- Fetch `GET /fleet` no padrão `useEffect`/`cancelled` canônico (igual Overview).
- Um `<MachineCard/>` (reuso do substrato) por máquina.
- Tabela DENSA de heartbeat: `machine_id`, `canonical_name`, `os_version`, `agentd_version`, `last_doctor` (Badge), `frescor` ("último sinal há Xh").
- Painel **version-drift por IGUALDADE DE STRING** (`detectDrift`): `Set<string>` de valores distintos por chave de `installed_versions`; drift = `Set.size > 1`. Zero parse numérico — `1.1.0` ≠ `1.36.0` é só "duas strings diferentes". Drift ⇒ Badge âmbar (`--status-warning`) + LABEL textual **"drift"** (cor nunca o único sinal).
- Frescor honesto e textual (`ageLabel`): remoto mostra idade real, nunca animação.

### Task 2 — Cofre-Espelho (`apps/cockpit/src/pages/CofreEspelho.tsx`) — checkpoint:human-verify
- Fetch `GET /vault` (metadata-only; o endpoint não tem campo `value`).
- Matriz var × project: `{project, var_name, presença, risk_tier, idade, committed}`. **Nenhum controle de UI lê/copia/escreve/revela um valor** — a tela é value-free por construção (a camada de dados não carrega o valor; `api_key` sem coluna correspondente — credential-isolation estrutural).
- Banner de **doutrina** no topo: "Espelho de metadados — o valor de um segredo jamais transita por aqui".
- **Empty-state CELEBRADO**: "Cofre íntegro — nada faltando, nada exposto" quando `missing==0 && exposed==0`.
- `risk_tier` critical ⇒ Badge com o **próprio texto do tier** como label (WCAG — cor nunca o único sinal).
- **Screenshot deferido ao orchestrator** (visual pass consolidada pós-W3, `frontend-visual-loop`): este executor não tem ferramenta de screenshot. Todos os critérios por exit-code estão satisfeitos; não houve espera por sinal "approved" (conforme escopo do orchestrator).
