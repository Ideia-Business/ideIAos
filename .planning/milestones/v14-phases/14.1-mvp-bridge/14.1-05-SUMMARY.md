## Decisions Made
- **Tape build-time, não runtime:** a história de `versions.lock` é imutável; o prebuild gera a fita e o gate re-deriva e compara — pega qualquer staleness sem rodar git no servidor loopback.
- **Gate reusa o build script (não um 2º reader):** via `FLIGHT_RECORDER_OUT` o gate escreve a re-derivação num `/tmp` sandbox e diffa contra `src/`. Um reader paralelo poderia driftar do render; reusar o mesmo elimina essa classe de bug.

### Auto-fixed Issues

**1. [Rule 1 - Bug] `deriveHost` capturava prosa como "host"**
- **Found during:** Task 1 (inspeção do JSON gerado)
- **Issue:** o regex inicial `\(([^()]+)\)\s*$` casava qualquer parêntese final, rotulando `(migração para get-shit-done-redux)` e `(pin pela instalacao real)` como `host` — fabricação adjacente (viola LAW vs INTERPRETED).
- **Fix:** endurecido para `\(([A-Za-z][A-Za-z0-9-]*)\)\s*$` (só token com forma de host). Prosa => `host: null` honesto.
- **Files modified:** apps/cockpit/scripts/build-flight-recorder.mjs
