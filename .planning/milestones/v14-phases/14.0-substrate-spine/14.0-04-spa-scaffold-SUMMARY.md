## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking] @types/node ausente para import.meta.url e path**
- **Found during:** Task 1 — `tsc -b` falhou com `Cannot find module 'path'` e `Cannot find name '__dirname'`
- **Issue:** Vite 7 + `type: "module"` em package.json requer ESM-safe — `__dirname` não existe em ESM; `path` precisa de `@types/node`
- **Fix:** (a) Substituído `path.resolve(__dirname, "./src")` por `fileURLToPath(new URL("./src", import.meta.url))` no vite.config.ts; (b) adicionado `@types/node` como devDependency; (c) adicionado `"types": ["node"]` no tsconfig.node.json
- **Files modified:** `apps/cockpit/vite.config.ts`, `apps/cockpit/tsconfig.node.json`, `apps/cockpit/package.json`
- **Commit:** `1fbcdcf`

**2. [Rule 3 - Blocking] `/// <reference types="vite/client" />` ausente — CSS import falhou em tsc**
- **Found during:** Task 1 — `tsc -b` falhou com `Cannot find module './index.css'`
- **Issue:** TypeScript não reconhece imports CSS sem o `vite/client` type reference
- **Fix:** Criado `apps/cockpit/src/vite-env.d.ts` com `/// <reference types="vite/client" />`
- **Files modified:** `apps/cockpit/src/vite-env.d.ts` (novo)
- **Commit:** `1fbcdcf`

**3. [Rule 2 - Missing critical] postcss.config.js e tsconfigs compostos adicionados**
- **Found during:** Task 1 — Tailwind 3 exige PostCSS config; Vite 7 requer `tsconfig.json` composto (`tsc -b`) com `tsconfig.app.json` + `tsconfig.node.json`
- **Issue:** Os 8 arquivos do plano não incluíam esses arquivos de config necessários para build funcionar
- **Fix:** Criados `postcss.config.js`, `tsconfig.json`, `tsconfig.app.json`, `tsconfig.node.json`
- **Files modified:** 4 arquivos novos além dos 8 do plano
- **Commit:** `1fbcdcf`
