## Decisions Made

1. **node:sqlite nativo (Node 24):** Não usar `better-sqlite3` — o plano 06 é o primeiro consumidor de SQLite. `node:sqlite` com `DatabaseSync` é nativo do Node 24 (sem npm install, sem build de módulo nativo, zero superfície supply-chain — T-14.0-06-SC).

2. **security-ledger NÃO ingerido (FIX-06C):** `.security/review-ledger.log` não ingerido nesta fase. Tabela `security_seal` é v14.2+. Carry-forward correto.

3. **actor_class em payload_json:** `commit_log` é v14.2+. Em v14.0, `actor_class` é materializado dentro de `machine_snapshot.payload_json -> $.commits[].actor_class`. Gate FIX-06D valida por `json_extract` contra DB real (não prosa no SUMMARY).

4. **gsd string-equality:** Nunca `parseInt`/`semver`/`localeCompare` para versão gsd — learning `version-reset-migration-semver-trap` (1.1.0 redux > 1.36.0 pré-redux: inversão silenciosa).

5. **machine-aliases.json:** Alias `192 -> Mac-mini-de-Gustavo` curado para resolver gotcha do substrato real (doc 73: hostname `192` é alias do Mac-mini).
